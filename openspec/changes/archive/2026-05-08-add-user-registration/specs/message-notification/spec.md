## ADDED Requirements

### Requirement: 注册成功后必须发布通知消息
系统 MUST 在用户注册成功后发布一条注册成功通知消息，以触发后续短信处理流程。

#### Scenario: 发布注册成功消息
- **WHEN** 用户注册成功并完成持久化写入
- **THEN** 系统必须向消息通道发布包含用户标识、用户名和手机号的通知消息

### Requirement: 短信发送必须提供占位实现
系统 MUST 为注册成功通知提供可运行的短信占位实现，在当前阶段以打印方式替代真实短信发送。

#### Scenario: 处理注册成功消息
- **WHEN** 短信通知处理器收到注册成功消息
- **THEN** 系统必须调用打印方法输出短信发送内容而不接入真实短信服务
