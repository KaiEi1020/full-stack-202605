## Why

当前简历工作台已经完成 REST 与 AI 评分重构，但导航与详情页交互仍存在明显体验问题：评分图表入口冗余、状态流转展示不清晰、详情页缺少更直观的多维度图形表达，同时部署链路还缺少镜像打包与远程启动脚本验证。现在需要收敛导航、修正详情页状态表现、补充雷达图展示，并把镜像与远程启动流程纳入同一变更中完成验证。

## What Changes

- 删除顶部导航中的评分图表 TAB，避免与简历详情页中的评分内容重复。
- 修复简历详情页中的状态流转展示问题，确保当前状态与可操作状态切换表现正确。
- 在简历详情页补充雷达图，用于展示技能匹配度、经验相关性、教育背景契合度与综合评分的图形化结果。
- 隐藏顶部导航中的用户注册 TAB，减少与当前招聘工作台主流程无关的入口暴露。
- 新增镜像打包与远程启动验证要求，要求项目能够完成镜像构建并验证 `bash ./scripts/start-remote.sh` 可执行。

## Capabilities

### New Capabilities
- `remote-runtime-validation`: 定义镜像打包与远程启动脚本验证要求

### Modified Capabilities
- `resume-score-visualization`: 将评分展示收敛到简历详情页，并补充雷达图要求
- `resume-rest-api`: 调整简历详情页交互相关要求，确保状态流转展示与操作结果一致

## Impact

- 影响 `frontend/` 中的导航、简历详情页、评分图表组件与相关交互逻辑
- 影响 `scripts/` 中的远程启动脚本验证流程
- 影响镜像构建与本地/远程运行验证方式
- 不改变现有 REST API 基础协议，但可能影响前端对状态更新结果的呈现方式
